## Algunas aclaraciones sobre los materiales 
### Tareas 6.2 y 6.3


Usan los imprimibles `clasificacion1.pdf` y `clasificacion2.pdf`. En cada uno de estos, la primera y segunda página son las láminas que toma cada pareja, y la tercera página, contiene todos los puntos que hay entre ambas láminas. También contiene este repositorio los archivos `clasificacion1.ggb` y `clasificacion2.ggb`, para facilitar al docente la creación de más imprimibles.

### Tarea 6.5
Imprimir una imagen `quienesquien.png` por cada alumno y alumna. Recomendamos recortarla en ocho partes, cada una con un animal distinto. Esto puede hacer más interactivo la creación del árbol de decisión.

### Tarea 7.3   
Usar la imagen `kmeans-cara.gif` para una ejecución más rápida de la tarea y el archivo `cara.ggb` si se considera interesante practicar más el algoritmo de $k$-medias.

### Tarea 7.4
Imprimir un archivo `cara.pdf`  por alumno y alumna.

### Tareas 9.3 y 10.3
Los archivos `.wxmx`usan el lenguaje SageMath y los `.ipynb` son archivos de Jupyter Notebook, y se usará en la tarea un kernel de python si el nombre del archivo contine `Python` y un kernel de `SageMath` si el nombre del archivo contiene `Sage`.    Si en el nombre del archivo pone `Docente`, contiene las soluciones, con intención de servir de ayuda al docente. En caso contrario, no contiene soluciones.
